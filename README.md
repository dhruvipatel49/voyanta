# Voyanta – AI-Powered Collaborative Travel Planner

Voyanta is an AI-powered travel planning platform developed during the **Breach 2026 Hackathon (3-day event)**. The platform simplifies trip planning by combining itinerary generation, collaboration, and expense management into a single system.

---

## 🚀 Project Overview

Voyanta allows users to input travel ideas in natural language (e.g., *“Plan a 4-day Goa trip”*), which are processed using AI to generate structured, personalized itineraries.

The system focuses on **practical and human-centered travel planning**, considering real-world constraints such as travel time, distance between locations, and user fatigue.

---

## 🧠 Key Features

* **AI-based itinerary generation** from natural language input
* **Human-centered planning** considering time, distance, and fatigue
* **Collaborative trip planning** with voting system
* **Wishlist feature** to save preferred locations
* **Expense tracking and splitting** (similar to Splitwise)
* **Route optimization** for efficient travel planning
* **Centralized trip dashboard** for managing plans

---

## 🛠️ Tech Stack

**Frontend:** React.js
**Backend:** Node.js, Express
**Database & Authentication:** Supabase
**AI Integration:** OpenAI API
**Data Source:** Travel dataset (attractions, restaurants, hotels)

---

## ⚙️ System Architecture

The platform follows a full-stack architecture:

* User inputs are handled through the frontend (React)
* Backend APIs (Node.js + Express) process requests
* OpenAI API generates structured itineraries
* Supabase manages user data, authentication, and trip storage
* Travel dataset is used to provide reliable recommendations

---

## 🎯 Problem Statement

Trip planning is often fragmented across multiple platforms, making it inefficient and time-consuming. Users struggle with:

* Creating structured itineraries
* Coordinating with friends
* Managing travel budgets
* Optimizing travel routes

Voyanta solves these challenges by providing a **single intelligent planning system**.

---

## 💡 Solution

Voyanta acts as an **AI-powered planning layer** that:

* Converts user intent into structured plans
* Enables group decision-making
* Optimizes travel routes
* Simplifies budget management
